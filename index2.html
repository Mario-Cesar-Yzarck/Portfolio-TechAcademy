Biblioteca do Polo

Site para a biblioteca do Polo UAB (em elaboração)

O projeto nasceu como um local onde os livros do Polo da UAB de Ecoporanga pudessem ser cadastrados e apresentados aos alunos. Com o tempo novas funcionalidades foram sendo acrescentadas, como formulário para contato e possibilidade de reserva de livros. Todavia o projeto ainda se encontra em elaboração. E a ideia é que possa ser feita uma aplicação web em que qualquer pessoa possa cadastrar seus livros, como uma biblioteca pessoal online.

Projeto Coordenadas

Página para obtenção de coordenadas (em elaboração)

Este projeto foi um pedido do meu irmão. Ele é engenheiro e a empresa que ele trabalha estava solicitando um local (página web) onde fosse possível identificar os clientes e apresentar um ponto de coordenada central. A ideia é criar um banco de dados e acessar as informações das empresas clientes já cadastradas e mostrar as informações sobre as mesmas nesta página. O projeto ainda não foi finalizado a previsão de encerramento e apresentação é para o final do ano de 2022.

OriginSix

Projeto elaborado no NLW da Rocketseat

Este projeto foi elaborado durante o evento NLW da Rocketseat. Foi um projeto importante para minha formação como dev por ser uma aplicação completa, "funcional". Pude aperfeiçoar conhecimentos em HTML, CSS e JavaScript e apesar de ser um conteúdo extenso senti um pouco como é trabalhar com prazos e tendo de levar em consideração várias etapas da construção da aplicação. Foi um dos primeiros projetos que fiz e pude ver resultados significativos.
